# Local Fonts

The site ships fonts locally under `public/fonts/`.

- `Almarai` handles Arabic and Latin UI text
- `Inter` handles Latin UI text
- `Kawkab Mono` handles monospace text, including Arabic-safe canvas rendering
- the app no longer loads Google Fonts or other third-party font servers at runtime

## Sources

- `Almarai` and `Inter` were vendored from Google Fonts
- `Kawkab Mono` was downloaded from https://makkuk.com/kawkab-mono/downloads/kawkab-mono-0.500.zip
